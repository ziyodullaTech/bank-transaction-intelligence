# Digital Bank Transaction Intelligence

An end-to-end near real-time Business Intelligence project that simulates
digital banking transactions and delivers actionable insights through
automated dashboards.

## Project Goal
To provide real-time visibility into transaction performance, customer behavior,
and potential risk indicators for banking stakeholders.

## Tech Stack
- PostgreSQL (Database)
- SQL (Data analysis & KPIs)
- Python (pandas, data generation & ETL)
- Power BI (Visualization & dashboards)
- GitHub (Version control & portfolio)
